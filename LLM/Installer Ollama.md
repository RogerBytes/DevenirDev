# Ollama

```bash
curl -fsSL https://ollama.com/install.sh | sh
```


on utilise l'aide avec

```bash
ollama --help
```

On installe un modèle

```bash
ollama pull gemma3:4b
```


Pour lister les modèles

```bash
ollama list
```

Pour lancer un modèle :

```bash
ollama run gemma3:4b
```
